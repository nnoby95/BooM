## Overview

### Purpose
Central dashboard to manage 30+ Tribal Wars accounts across multiple worlds with:
- Real-time account monitoring
- Individual and bulk operations
- Attack alerts
- Building and recruitment automation via templates

### Key Principles
- **Desktop-first** design
- **Tribal Wars visual style** (colors, textures, icons)
- **Real-time updates** via WebSocket
- **Template-based automation** for building and recruitment
- **Single village per account** (no multi-village support needed)

#### How Building Template Execution Works

1. **Parse template** into ordered array of steps
2. **Track progress** per account (which step they're on)
3. **When executing for an account:**
   - Get account's current step index
   - Check if building already at/above target level â†’ skip to next step
   - Check if resources sufficient â†’ execute or wait
   - Check if prerequisites met â†’ execute or error
   - On success â†’ increment step index
4. **MINES X** expands to: `wood X; stone X; iron X`
5. **Auto-rules** can inject storage/farm upgrades when needed
